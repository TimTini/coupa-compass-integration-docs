Expense Attendee Type API

Association

This resource is associated with Expense Lines API via Expense Attendee API.

Warning:

Deactivating the Employee, Company Users expense attendee type is not recommended, because it will result in a loss of search functionality for Coupa Mobile users attempting to find employees for inclusion on an expense report.
