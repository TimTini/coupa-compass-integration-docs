# View Treasury API Documentation

You can access the Treasury API documentation directly from the Treasury Management application.

- Click **More** in the **User Toolbar**. ![](https://compass.coupa.com/_dita_/en-us/documentation/plat/integ/treasury_integrations/images/image1.png)

- From the menu, select **API documentation**. This opens the API documentation in a new tab. Select an API integration from the list to expand it.

![](https://compass.coupa.com/_dita_/en-us/documentation/plat/integ/treasury_integrations/images/image2.png)
