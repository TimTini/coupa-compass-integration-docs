Business Entities API (/business_entities)

Use the budget line API to create and update budget lines that you have associated with your accounts.

The URL to access business entities is: https://<instance>/api/business_entities

See Integration Best Practices for more info.

Actions

The Business Entities API allows you to:

Verb 	 Path 	 Action 	 Description
GET 	 /api/business_entities 	query 	Query business entities
POST 	 /api/business_entities 	create 	Create business entity
GET 	 /api/business_entities/:id 	show 	Show business entity
PATCH 	 /api/business_entities/:id 	update 	Update business entity
PUT 	 /api/business_entities/:id 	update 	Update business entity
