UserGroups

The User Groups quick integration import allows Risk Assess users to directly upload spreadsheets via the HTTPS protocol. This allows customers to perform inbound imports in an automated manner without the need for an Engagement interface.

Note:
- This import adds an existing user to an existing UserGroup.
- You cannot remove UserGroups using this import.
- This import won't create a new UserGroup.

Post URL

https://<CRA_TENANT>.hiperos.com/QuickIntegration/UserGroups
