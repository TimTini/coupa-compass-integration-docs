Tokens in the Coupa Supplier Portal

Cccess a selected invoicing functionalities with a token-based system.

 The Coupa Supplier Portal (CSP) has a token-based system for specific advanced invoicing features. This provides greater flexibility, cost control, and visibility into supplier usage of invoicing functionalities.

Invoicing features with token cost

 Suppliers consume tokens for each use of the selected advanced features: 

  • InvoiceSmash: for each invoice uploaded.
    For more information, see  Submit non-PO PDF invoices using the InvoiceSmash AI extraction tool.
  • Invoice synchronization with NetSuite and QuickBooks: for each invoice line synced.
    For more information, see QuickBooks Online & NetSuite integration.
  • Coupa Supplier Portal REST API: 
    For more information, see Coupa Supplier Portal REST API.

Token costs are determined by the type and size of the transaction. 

How to review token use and costs

Suppliers with Coupa Advanced and a Coupa Advanced Trial receive a free amount of tokens upon activation of this feature. To review how many tokens you have and how you can use them: 

  1. Log in to the CSP.
  2. Select the tab Business Profile > Suscriptions.
    The window displays information about your suscription services, including Coupa Advanced, Premium Support and Coupa Verified. 
  3. Review the first tile of the page called Token Usage. 
    • Hover over the information icon to see the Token Usage Guide to read more about the costs of transactions. 
    • Select the View Detail option to see insights about your token usage.
    • Select the Purchase Token button to purchase additional tokens and review their cost.
