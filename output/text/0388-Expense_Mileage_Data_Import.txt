Expense Mileage Data Import

Overview

The expense mileage data import process reads files from the ./Incoming/ExpenseMileageData/ in the SFTP. These files will be moved to the archive folder located at ./Incoming/Archive/ExpenseMileageData/ before being processed in Alpha-Numeric order.
