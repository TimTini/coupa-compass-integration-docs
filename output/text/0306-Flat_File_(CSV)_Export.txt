Flat File (CSV) Export

Coupa makes standard transactional data available for download as part of the standard Coupa Platform license. Data records are exported as flat files in a CSV (comma separated value) format. You can inspect the CSV records by exporting a template and you can change what fields are exported and the order of those fields with integration formats.

Note

All lines in outbound CSV files have the same number of columns. This may mean that some lines will have blank columns.

You can also download the flat file formats as an excel version for mapping out your integrations. See API and CSV Download Formats.
