Purposes API

Use the purposes API to query, create, or update personal or supplier remit-to address information.

Associations

This API is associated with Address API, Contact API and Suppliers API.

Use the purposes API to query, create, or update the purpose of supplier contact information.

Elements

The following elements are available for the Purposes API:
