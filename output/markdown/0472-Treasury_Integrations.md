# Treasury Integrations

This section contains documentation for integrations used with the Treasury Management application.

Treasury integrations include

- [Treasury APIs](https://compass.coupa.com/en-us/products/product-documentation/integration-technical-documentation/treasury-integrations/treasury-apis) that you use directly from the Treasury Management application.

- [sFTP integrations](https://compass.coupa.com/en-us/products/product-documentation/integration-technical-documentation/treasury-integrations/treasury-sftp-integrations) for CSV imports and exports that you access from the Coupa Core platform.

- [Treasury CSV Format Descriptions (classic)](https://compass.coupa.com/en-us/products/product-documentation/integration-technical-documentation/treasury-integrations/treasury-csv-format-descriptions-(classic)) for the classic Treasury Management interface.

- [Treasury APIs](https://compass.coupa.com/en-us/products/product-documentation/integration-technical-documentation/treasury-integrations/treasury-apis)

Retrieve and update Treasury Management data such as cash flows and account balances.

- [Treasury sFTP Integrations](https://compass.coupa.com/en-us/products/product-documentation/integration-technical-documentation/treasury-integrations/treasury-sftp-integrations)

Use sFTP for Treasury CSV import and export integrations. Set up and manage your integrations using Coupa Core.

- [Treasury CSV Format Descriptions (classic)](https://compass.coupa.com/en-us/products/product-documentation/integration-technical-documentation/treasury-integrations/treasury-csv-format-descriptions-(classic))
