# US Government Cloud

IP addresses that should be whitelisted by US suppliers and customers.

Use this list of Coupa Application sandbox and production
environment IP addresses for customers with restriction to access
outside corporate networks and for the supplier setup with cXML
document exchange.

![](https://compass.coupa.com/DITARoot/icons/important.png)
Note:

SFTP and Outbound email use the same IP address ranges listed below.

## IP Addresses

All addresses below should be whitelisted as data could come
from any one of these IPs.

| **Address Ranges** | **Date Added** |
| --- | --- |
| 96.127.126.128 through 96.127.126.159 | Feb 2023 |
| 3.32.212.128 through 3.32.212.191 | Feb 2021 |
| 96.127.126.150<br>96.127.126.151 | Oct 2018 |
