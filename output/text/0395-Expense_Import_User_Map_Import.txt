Expense Import User Map Import

Overview

The Expense Import User Map Import process read files from the ./Incoming/ExpenseImportUserMaps/ in the SFTP. These files will be moved to the archive folder located at ./Incoming/Archive/ExpenseImportUserMaps/ before being processed in Alpha-Numeric order.

Possible values Corp Card Integration Name Must match the name of the integration on the Setup > Integrations page. Account Type For Employee Id mapping, use EMPLOYEE_ID. When mapping with account number, use one of AMEX, VISA, or MCC. Reimbursement Type Corporate Bill; Individual Bill
